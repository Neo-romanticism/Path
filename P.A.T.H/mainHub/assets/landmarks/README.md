# University Landmark Models (.glb)

Place high-quality building models here to replace procedural fallback geometry.

## Expected files

- `snu.glb` for 서울대학교
- `yonsei.glb` for 연세대학교
- `korea.glb` for 고려대학교
- `kaist.glb` for 카이스트
- `postech.glb` for 포항공과대학교
- `skku.glb` for 성균관대학교
- `hanyang.glb` for 한양대학교
- `chungang.glb` for 중앙대학교
- `kyunghee.glb` for 경희대학교
- `sogang.glb` for 서강대학교
- `ewha.glb` for 이화여자대학교
- `pusan.glb` for 부산대학교

## Notes

- Coordinate recommendation: model front along +Z in your DCC tool.
- Keep model origin near ground center for easy placement.
- Target poly budget: 20k-80k triangles per building.
- Use baked textures/PBR materials for best quality.
- If a file is missing, the app automatically uses procedural fallback geometry.
